bodittoolbox-api

바딧툴박스 API 서버

툴박스 프로그램, 대시보드에서 사용하는 API서버입니다.

DEV(개발, 테스트) 주소

https://bodittoolbox-api-dev.herokuapp.com/v1

PROD(상용) 주소

https://bodittoolbox-api-prod.herokuapp.com/v1

# compile typescript
tsc

# heroku-cli를 설치합니다.
brew tap heroku/brew && brew install heroku

# remote(dev)를 추가해줍니다.
heroku git:remote -a bodittoolbox-api-dev
git remote rename heroku heroku-dev

# remote(prod)를 추가해줍니다. 
heroku git:remote -a bodittoolbox-api-prod
git remote rename heroku heroku-prod

# heroku develop 브랜치는 bodittoolbox-api-dev 앱에 push 해줍니다.
git push heroku-dev develop:main

# heroku main 브랜치는 bodittoolbox-api-prod 앱에 push 해줍니다.
git push heroku-prod main:main
