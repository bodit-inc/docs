Called whenever the object needs to paint. The given Canvas has its coordinate space configured such that the origin is at the top left of the box. The area of the box is the size of the size argument.

Paint operations should remain inside the given area. Graphical operations outside the bounds may be silently ignored, clipped, or not clipped. It may sometimes be difficult to guarantee that a certain operation is inside the bounds (e.g., drawing a rectangle whose size is determined by user inputs). In that case, consider calling Canvas.clipRect at the beginning of paint so everything that follows will be guaranteed to only draw within the clipped area.

Implementations should be wary of correctly pairing any calls to Canvas.save/Canvas.saveLayer and Canvas.restore, otherwise all subsequent painting on this canvas may be affected, with potentially hilarious but confusing results.

To paint text on a Canvas, use a TextPainter.

To paint an image on a Canvas:

  1. Obtain an ImageStream, for example by calling ImageProvider.resolve on an AssetImage or NetworkImage object.

  2. Whenever the ImageStream's underlying ImageInfo object changes (see ImageStream.addListener), create a new instance of your custom paint delegate, giving it the new ImageInfo object.

  3. In your delegate's paint method, call the Canvas.drawImage, Canvas.drawImageRect, or Canvas.drawImageNine methods to paint the ImageInfo.image object, applying the ImageInfo.scale value to obtain the correct rendering size.

@override
void paint(Canvas canvas, Size size) {
  var linePaint = Paint()
    ..color = lineColor
    ..strokeWidth = 2
    ..style = PaintingStyle.stroke;
  var backgroundPaint = Paint()
    ..color = backgroundColor
    ..strokeWidth = 0
    ..style = PaintingStyle.fill;

  final linePath = Path();
  final backgroundPath = Path();

  double totalColumnWidth = columns.fold(0, (value, element) {
    return value + element.width;
  });
  var width = isFixed ? fixedColWidth : totalColumnWidth + 3 + 3 * 10 + 1 * 2;
  var height = 36.0;
  double startOffsetX = -7;
  if (isFixed) startOffsetX = 7;
  backgroundPath.moveTo(0, 0);
  backgroundPath.lineTo(width, 0);
  backgroundPath.lineTo(width, height);
  backgroundPath.lineTo(0, height);
  backgroundPath.lineTo(0, 0);
  canvas.drawPath(backgroundPath, backgroundPaint);

  linePath.moveTo(startOffsetX, -height);
  if (isFixed) {
    linePath.lineTo(startOffsetX + width + 3, -height);
    linePath.lineTo(startOffsetX + width + 3, -1);
    linePath.lineTo(startOffsetX, -1);
  } else {
    linePath.lineTo(startOffsetX + width - 2 * 2, -height);
    linePath.lineTo(startOffsetX + width - 2 * 2, -1);
    linePath.lineTo(startOffsetX, -1);
  }
  if (isFixed) linePath.lineTo(startOffsetX, -height - 1);
  canvas.drawPath(linePath, linePaint);

  if (!isFixed) {
    // double offsetX = columns[1].width;
    double offsetX = columns[1].width;
    final double fontSize = 14;
    for (var i = 0; i < columns.length; i++) {
      if (i <= 1) continue;

      String text = (values[i] / 60).round().toString();

      String suffix = "분";

      // if (i == 5) {
      //   text = values[i].round().toString();
      // } else if (i == 8) {
      //   text += "";
      // } else if (i == 9 || i == 10) {
      //   text = "";
      // } else

      if (i == 5 || i == 8) {
        // 5=포유자극, 8=기침
        text = values[i].round().toString();
        suffix = "회";
      }

      // 활동량
      if (i == 9) {
        text = (values[i] * NumericConstants.activityScale).round().toString();
        suffix = "";
      }

      // 10=소화기, 11=호흡기
      if (i == 10 || i == 11) {
        text = values[i] == -1 ? "-" : values[i].round().toString();
        suffix = "";
      }

      // BCS
      if (i == 12) {
        if (values[i] == 0)
          text = "좋음";
        else if (values[i] == 1)
          text = "보통";
        else if (values[i] == 2)
          text = "나쁨";
        else
          text = "-";
        suffix = "";
      }

      text += suffix;
      // String text = cowData.data.first.cowDataHour.first.digestive.value.toString();

      TextSpan span = new TextSpan(text: text, style: TextStyle(color: Colors.black, fontSize: fontSize));
      TextPainter tp = new TextPainter(text: span, textDirection: TextDirection.ltr, textAlign: TextAlign.center);
      tp.layout(minWidth: columns[i].width);
      tp.paint(canvas, new Offset(offsetX, height / 2 - fontSize / 2));
      tp.paint(canvas, new Offset(offsetX, height / 2 - fontSize / 2));
      offsetX += columns[i].width + 20;
      // offsetX += (i == 1 || i == 6) ? 1 : 3;
    }
  }
}
