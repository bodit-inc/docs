바딧툴박스

CCTV 영상 태깅 작업을 위한 내부 프로그램입니다.

개발모드

// electron과 react를 함께 실행합니다.
npm run electron-dev

// react만 실행합니다. electron-renderer.js에서 electron-renderer로 타겟팅이 되어있기 때문에 웹 브라우저상에서는 실행이 되지 않습니다. 
npm run start

// electron만 실행합니다.
npm run electron

배포하는 방법

npm run dist 명령어를 입력하면 dist/ 디렉토리에 .exe, .dmg 파일이 생성됩니다.
